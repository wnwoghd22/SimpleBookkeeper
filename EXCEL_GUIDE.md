# 샘플 XLSX 파일 생성 가이드

이 문서는 복식부기 가계부 앱에서 사용할 수 있는 샘플 XLSX 파일을 만드는 방법을 설명합니다.

## 방법 1: 웹 앱에서 직접 생성

1. 웹 앱 실행 후 몇 개의 거래를 입력
2. "엑셀 내보내기" 버튼 클릭
3. 생성된 파일을 템플릿으로 사용

## 방법 2: Excel/Google Sheets에서 직접 작성

### 필수 열 구조

**시트 이름**: 거래내역 (또는 임의의 이름, 첫 번째 시트가 자동으로 읽힘)

| 열 이름 | 타입 | 설명 | 예시 |
|---------|------|------|------|
| 날짜 | 텍스트 | YYYY-MM-DD 형식 | 2025-01-15 |
| 적요 | 텍스트 | 거래 설명 | 월세, 점심식사, 급여 등 |
| 차변계정 | 텍스트 | 차변 계정명 | 식비, 은행예금 등 |
| 차변금액 | 숫자 | 차변 금액 | 50000 |
| 대변계정 | 텍스트 | 대변 계정명 | 현금, 급여 등 |
| 대변금액 | 숫자 | 대변 금액 | 50000 |

## 방법 3: Python으로 샘플 파일 생성

```python
import pandas as pd
from datetime import datetime, timedelta
import random

# 샘플 데이터 생성
transactions = []

# 급여
transactions.append({
    '날짜': '2025-01-01',
    '적요': '1월 급여',
    '차변계정': '은행예금',
    '차변금액': 3000000,
    '대변계정': '급여',
    '대변금액': 3000000
})

# 현금 인출
transactions.append({
    '날짜': '2025-01-02',
    '적요': '현금 인출',
    '차변계정': '현금',
    '차변금액': 500000,
    '대변계정': '은행예금',
    '대변금액': 500000
})

# 일상 지출들
expenses = [
    ('식비', ['편의점', '점심식사', '저녁식사', '커피', '간식'], 5000, 30000),
    ('교통비', ['지하철', '버스', '택시'], 1500, 15000),
    ('통신비', ['휴대폰 요금', '인터넷 요금'], 50000, 60000),
    ('문화생활비', ['영화', '책', '공연'], 10000, 50000),
]

start_date = datetime(2025, 1, 3)
for i in range(20):
    category, descriptions, min_amt, max_amt = random.choice(expenses)
    date = start_date + timedelta(days=i)
    amount = random.randint(min_amt, max_amt)
    
    transactions.append({
        '날짜': date.strftime('%Y-%m-%d'),
        '적요': random.choice(descriptions),
        '차변계정': category,
        '차변금액': amount,
        '대변계정': '현금',
        '대변금액': amount
    })

# DataFrame 생성 및 저장
df = pd.DataFrame(transactions)
df.to_excel('sample_transactions.xlsx', sheet_name='거래내역', index=False)

print(f"샘플 파일 생성 완료: {len(transactions)}개 거래")
```

## 데이터 검증 규칙

앱에서 XLSX 파일을 불러올 때 다음을 확인합니다:

1. **필수 필드**: 모든 행에 날짜, 적요, 차변계정, 차변금액, 대변계정, 대변금액이 있어야 함
2. **날짜 형식**: YYYY-MM-DD 형식 권장
3. **금액**: 0보다 큰 숫자여야 함
4. **차변/대변 일치**: 각 거래의 차변금액 = 대변금액 (앱에서 자동 검증)

## 주의사항

- 엑셀에서 날짜를 입력할 때 날짜 형식이 아닌 텍스트로 입력 (YYYY-MM-DD)
- 금액에 쉼표나 통화 기호를 넣지 말 것 (순수한 숫자만)
- 한글 계정명은 위의 목록과 정확히 일치해야 함
- 첫 번째 시트의 데이터만 불러와짐
